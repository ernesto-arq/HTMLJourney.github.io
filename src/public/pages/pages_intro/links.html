Geração de Links

Em HTML, a geração de links é realizada pela tag <a href="url">texto do link</a>. Por exemplo:


Este é um link para a seção de parágrafos!

A seguir, apresentamos como fazer uma referência a uma seção específica. Por exemplo, utilizando um link de salto, é possível navegar entre seções, como nas regiões da página a seguir:

<a id="section1">>Este é um exemplo de como criar um link de salto, ou seja Go To</a>
O link de salto irá redirecionar para <a href="#section1"></a> Ir para a Seção 1
