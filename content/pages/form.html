Formulários para HTML

Um formulário em HTML é utilizado pela tag from. Dentro da tag são colocados vários tipos de campos de entrada, como caixas de texto, botões de rádio, caixas de seleção e menus suspensos. Cada tipo de entrada é considerado um tipo de input para coletar diferentes tipos de dados do usuario.

Elementos de um formulário:

  • <input>Usado para criar vários tipos de campos de entrada, como texto, senha, número, e-mail, etc.
  • <textarea> Fornece uma área maior para texto, útil para comentários ou mensagens mais extensas.
  • <select> e <option> Criam um menu suspenso de onde o usuário pode selecionar uma opção.
  • <button> Usado para enviar o formulário, resetar o formulário, ou simplesmente atuar como um botão para outras funcionalidades.
  • <label> Fornece etiquetas para os campos de entrada, melhorando a acessibilidade e a usabilidade.

Atributos Importantes

  • <action> Especifica o URL do servidor onde os dados do formulário devem ser enviados.
  • <method> Define o método HTTP usado para enviar o formulário. Os métodos mais comuns são GET e POST.
  • <GET> Anexa os dados do formulário ao URL, sendo adequado para pesquisas e solicitações não-confidenciais.
  • <POST> Envia os dados do formulário no corpo da solicitação, adequado para dados sensíveis ou grandes quantidades de informação.

Quando um usuário preenche o formulário e clica em um botão de envio, o navegador coleta todos os dados dos campos, constrói uma mensagem de solicitação de acordo com o método especificado (GET ou POST), e a envia ao servidor indicado no atributo action.
