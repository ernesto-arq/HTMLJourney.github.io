Aplicações da div em HTML

A tag <div> em HTML é um elemento de bloco usado para agrupar outros elementos HTML e aplicar estilos ou scripts a esses elementos como um único bloco. É frequentemente usada como um "container" para outros conteúdos, pois não tem significado semântico específico. A <div> é muito útil para organizar a estrutura de uma página, permitindo um controle mais flexível do layout por meio de CSS.

O Código a seguir é colocado dentro da head do corpo em HTML e apresentará uma formatação específica para cada div dentro da página:

        <style>
            #header {
                background-color: #d30808;
                padding: 20px;
                text-align: center;
                font-size: 24px;
            }
            #main {
                background-color: #e9ecef;
                padding: 20px;
                margin-top: 10px;
            }
            #footer {
                background-color: #dee2e6;
                padding: 10px;
                text-align: center;
                margin-top: 10px;
            }
        </style>
        
        <div id="header">Jornal de Notícias ABC</div>
        <div id="main">
            <div>Notícias</div>
            <div>Essas foram as notícias de hoje! Por Fulano de Tal. Lorem ipsum dolor sit amet consectetur, adipisicing elit. Vitae sapiente repellendus in 
            animi blanditiis, expedita praesentium possimus iusto, adipisci iste totam optio quibusdam aliquid minus harum debitis voluptate fuga doloremque! </div>
        </div>
        <div id="footer">Essas foram as notícias de hoje! Por Fulano de Tal.</div>
