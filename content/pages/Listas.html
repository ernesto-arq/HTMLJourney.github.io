Listas em HTML

Listas em HTML são ferramentas versáteis para organizar conteúdo de forma clara e estruturada, seja enumerando etapas, agrupando itens relacionados ou simplificando a navegação dentro de um documento.

As listas em HTML são divididas em três categorias: listas não ordenadas (<ul>), listas ordenadas (<ol>) e elementos de lista (<li>).

Exemplo de lista não ordenada:

Uma lista não ordenada é usada para agrupar itens sem uma ordem específica. Abaixo está a sintaxe e um exemplo:

<ul>
<li>Isso é um item.</li>
<li>Esse é o segundo item.</li>
</ul>
  • Isso é um item.
  • Esse é o segundo item.

Exemplo de lista ordenada:

Uma lista ordenada é usada para itens que seguem uma ordem sequencial. Abaixo está a sintaxe e um exemplo:

<ol>
<li>Isso é um item.</li>
<li>Esse é o segundo item.</li>
</ol>
  1. Isso é um item.
  2. Esse é o segundo item.
