Listas em HTML

As listas em HTML são dividida em três categorias, listas não ordenadas, <ul> listas ordenadas <ol> e elementos da lista <li>
Por exemplo uma lista não ordenada:

Lista de itens a seguir:

<ul>
    <li> Isso é um item. <li>
    <li> Esse é o segundo item. <li>
<ul>
Desta forma, a lista não ordenada assume a seguinte forma:

Lista de itens a seguir:

  • Isso é um item.
  • Esse é o segundo item.
Por exemplo uma lista ordenada:

Lista de itens a seguir:

<ol>
    <li> Isso é um item. <li>
    <li> Esse é o segundo item. <li>
<ol>
Desta forma, a lista não ordenada assume a seguinte forma:

Lista de itens a seguir:

  1. Isso é um item.
  2. Esse é o segundo item.
