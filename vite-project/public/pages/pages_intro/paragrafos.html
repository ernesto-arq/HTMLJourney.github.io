Parágrafo

A seguir, detalhamos sobre formas de utilizar o código HTML para a criação de parágrafos, assim como para a formatação de texto, abordando espaçamento e semântica.

    Espaçamento

  • O elemento <p> serve para adicionar e/ou forçar um espaço em um texto;
  • O elemento <br> serve para a formatação de texto, colocando uma linha após terminar um texto específico.
Além disso, esse tipo de formatação pode ser adicionada via estilo de CSS, ou seja, alterando a formatação do arquivo CSS é possível passar uma sequência de instruções que formatará a página, inclusive a estruturação de texto.
