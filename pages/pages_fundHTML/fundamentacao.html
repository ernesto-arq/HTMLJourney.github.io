Entendendo as Tags HTML e sua Estrutura

Neste guia, abordaremos o funcionamento, significado e aplicações de tags HTML e como elas compõem a estrutura de documentos web. Vamos começar com a anatomia básica de uma tag HTML:
<tag>Conteúdo da tag</tag> - Essa é a forma mais básica de uma tag, onde "tag" representa o tipo de elemento e o conteúdo entre as tags é o que será exibido no navegador.

Tags também podem ser auto-fecháveis, o que significa que não necessitam de uma tag de fechamento. Por exemplo:
<tag /> - Conhecidas como tags auto-fecháveis, são interpretadas pelos navegadores como completas em si mesmas.

Além disso, uma tag pode conter atributos que definem suas propriedades ou comportamento, como por exemplo:
<tag atributo="valor"> - Neste exemplo, a tag possui um atributo chamado "atributo" com o valor "valor".
Uma tag pode incluir múltiplos atributos:
<tag atributo1="valor1" atributo2="valor2" atributo3="valor3">

É possível também especificar atributos e conteúdo dentro de uma mesma tag:
<tag atributo1="valor1" atributo2="valor2">Conteúdo da tag</tag>

Algumas tags geram elementos interativos diretamente, como:
<input type="color">
<input type="date">
<input type="checkbox">
<input type="email" placeholder="joao@email.com.br">
<input type="password" placeholder="essaEumaSenha">

Tags podem ser aninhadas, criando uma hierarquia de elementos, como no exemplo a seguir com listas:
<ul> <li>Sublista1. Algum texto.</li> <li>Sublista2. Mais algum texto.</li> </ul>
Cada tag <li> dentro do <ul> representa um item da lista, demonstrando como as tags podem conter outras tags, formando uma estrutura de elementos aninhados.
