Inserção de Imagens em HTML

Este guia explica como inserir e configurar imagens em uma página web usando a tag <img>. Esta tag possui atributos essenciais como 'src', 'alt' e 'title', que melhoram a usabilidade e acessibilidade do conteúdo.

Descrição dos Atributos:

  • src: Define o caminho do arquivo de imagem. Por exemplo, 'src="../imgs/dog.jpg"' aponta para uma imagem específica no diretório 'imgs'.
  • alt: Fornece um texto alternativo que é exibido quando a imagem não pode ser carregada. Este atributo é crucial para a acessibilidade, ajudando leitores de tela a descrever a imagem. Exemplo: 'alt="Descrição Alternativa"'
  • title: Oferece uma descrição adicional que aparece como uma dica de ferramenta quando o usuário passa o mouse sobre a imagem. Exemplo: 'title="Golden Retriever"'

Exemplo de uso básico da tag <img>:

<img src="../imgs/dog.jpg" alt="Descrição Alternativa" title="Golden Retriever">

Associando Imagens com Links:

Para criar um link clicável em uma imagem, envolva a tag <img> com a tag <a>:

<a href="../../index.html"><img src="../imgs/dog.jpg" alt="Descrição Alternativa" title="Golden Retriever" ></a>

Aplicando Estilos CSS em Imagens:

Use CSS para estilizar imagens. No exemplo abaixo, aplicamos um estilo que define a largura, a altura automática e a borda da imagem:

<img src="../imgs/dog.jpg" alt="Descrição Alternativa" class="imagem-estilizada">
