Imagens em HTML

A seguir, apresentamos como criar referências a imagens em HTML. Para inserir imagens, a tag utilizada é <img>. Essa tag é usada para inserir uma imagem em uma página web. Ela possui os seguintes atributos: 'src', 'alt' e 'title'.

Descrição dos Atributos:
  • src: Especifica o caminho para a imagem.
  • alt: Fornece um texto alternativo para a imagem, que será exibido se a imagem não puder ser carregada. Isso também é importante para acessibilidade.
  • title: Oferece uma descrição da imagem quando o usuário passa o mouse sobre ela.

    • <img src="../imgs/dog.jpg" alt="Descrição Alternativa" title="Golden Retriever" >
Imagem com associação de um link:
<a href="../../index.html">
    <img src="../imgs/dog.jpg" alt="Descrição Alternativa" title="Golden Retriever" >
</a>



Imagem com estilo em CSS
<style>  .imagem-estilizada{
    width:200px;
    height:auto;
    border:2px solid #000;
}
</style>
<img src="../imgs/dog.jpg" alt="Descrição Alternativa" class="imagem-estilizada">
