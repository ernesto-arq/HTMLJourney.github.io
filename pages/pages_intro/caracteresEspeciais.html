Caracteres Especiais HTML

Este guia apresenta uma lista de caracteres especiais comumente usados em HTML. Utilize estas referências para incorporar símbolos específicos em seus documentos HTML e entender melhor a sintaxe desta linguagem.

  • Espaço não-separável: &nbsp; - Utilizado para inserir um espaço que não pode ser separado por quebras de linha automáticas.
  • Menor que: &lt; - Representa o sinal de "menor que" (<), usado para evitar conflitos com tags HTML.
  • Maior que: &gt; - Representa o sinal de "maior que" (>), usado para evitar conflitos com tags HTML.
  • Aspas Simples: &apos; ou &#39; - Usado para representar aspas simples em conteúdo HTML, evitando conflitos com a marcação.
  • Aspas Duplas: &quot; ou &#34; - Usado para representar aspas duplas em conteúdo HTML, evitando conflitos com a marcação.
Símbolos Matemáticos:
  • Sinal de adição: &plus; - Representa o sinal de adição (+).
  • Sinal de subtração: &minus; - Representa o sinal de subtração (−).
  • Sinal de multiplicação: &times; - Representa o sinal de multiplicação (×).
  • Sinal de divisão: &equals; - Deveria ser &divide;, representando o sinal de divisão (÷).
  • Mais ou menos: &plusmn; - Representa o sinal de mais ou menos (±).
Símbolos Diversos:
  • Cópia de direitos autorais: &copy; ou &#169; - Usado para indicar direitos autorais.
  • Marca Registrada: &reg; ou &#174; - Usado para indicar uma marca registrada.
  • Marca de Serviço: &trade; ou &#8482; - Usado para indicar uma marca de serviço.
  • Sinal de parágrafo: &para; ou &#182; - Usado para indicar um novo parágrafo ou seção.
  • Sinal de seção: &sect; ou &#167; - Usado para indicar uma seção em um documento.
    Caracteres para formatação de texto:
  • Hífen/não separável: &shy; - Um hífen opcional que só é exibido no final de uma linha para auxiliar na quebra de palavras.
